一个快速发布Note到Gridea的Chrome插件

📝 简介
GrideaPush 是一个Chrome扩展程序，旨在简化Gridea博客用户的写作流程。通过此扩展，您可以直接从浏览器快速发布笔记、摘录或灵感到Gridea博客平台，无需切换到Gridea客户端。

✨ 功能特点

🚀 一键发布Note到Gridea

🔍 侧边栏模式，方便边浏览边记录
